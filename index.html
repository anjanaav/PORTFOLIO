Relax On Error is a Visual Studio Code extension created to improve the developer experience by making coding errors feel less stressful and more manageable. Smart Meeting Minutes Generator is an intelligent meeting automation system designed to join online meetings, record conversations, convert speech to text, identify speakers, and automatically generate structured meeting minutes. The main goal of the project is to reduce the manual effort required in documenting meetings by automatically producing a clear summary of discussions, decisions, action items, and deadlines.
